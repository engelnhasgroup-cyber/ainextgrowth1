     3‚Üí## Project Overview
     4‚ÜíA professional, ultra-fast, mobile-first AI Prompt & Skill Library targeting 2026+.
     5‚Üí- Autonomous AI Agent generates 100 trending prompts + 100 trending skills daily.
     6‚Üí- Every item ships as a "Trinity Bundle": 3 Markdown files (The Prompt/Skill, The Workflow, The Target Audience).

     9‚Üí- Dark-mode, futuristic UI. Single visible route `/` (views via state + modals).
